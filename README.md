# TelegramForwarder
Telegram Channel Or Group Forwarder Bot (CLI)

### Install :

````
git clone https://github.com/hesamz3090/TelegramForwader
cd TelegramForwader/
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
````

### Usage :
First You Need To Fill config.py by Your Data .

````
python main.py
````
